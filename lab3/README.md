# Lab 3 - GameVerse: Gaming Website with Bootstrap 5

## 🎮 Project Overview
**GameVerse** is a modern, responsive gaming platform website built with Bootstrap 5. Features stunning gaming aesthetics, smooth animations, and full responsiveness across all devices.

## 📋 Objective
Design and build a **responsive gaming website** using **Bootstrap 5** framework.

## 🎯 Task
Create a responsive gaming platform website (landing page) that works perfectly on mobile, tablet, and desktop devices.

## ✅ Requirements Met

### 1. Bootstrap Grid System ✓
- ✅ Fully responsive grid layout throughout
- ✅ Container, row, and col-* classes used extensively
- ✅ Responsive breakpoints: mobile (< 576px), tablet (768px), desktop (992px+)
- ✅ 6 game cards in 3-column layout (desktop) / 2-column (tablet) / 1-column (mobile)

### 2. Bootstrap Components Used (10+) ✓
1. **Navbar** - Responsive navigation with gaming theme and glow effects
2. **Carousel** - Hero section with 3 gaming slides
3. **Cards** - 6 game cards with ratings and badges
4. **Buttons** - Gaming-styled buttons with hover effects
5. **Forms** - Contact form with dropdown and validation
6. **Accordion** - About section with collapsible gaming stats
7. **Progress Bars** - Animated performance/community stats
8. **Modal** - Featured games and tournaments popup
9. **Badges** - Multiple badges for game categories and stats
10. **Bootstrap Icons** - Gaming icons throughout

### 3. Required Sections ✓
- ✅ **Responsive Navbar** - Fixed gaming nav with smooth scroll
- ✅ **Hero Section** - Full-screen carousel with 3 slides
- ✅ **Content Sections** - About, Games, Features sections
- ✅ **Contact Form** - Fully styled gaming contact form
- ✅ **Footer** - Gaming-themed footer with social links

### 4. Responsive Design ✓
- ✅ **Mobile** (320px+) - Optimized touch interface
- ✅ **Tablet** (768px+) - Enhanced layout
- ✅ **Desktop** (1024px+) - Full gaming experience

### How to Add Your Images/GIFs
1. Save your images to a folder: `/lab3/assets/images/`
2. Replace icon placeholders in HTML with:
   ```html
   <img src="assets/images/your-game.gif" class="img-fluid" alt="Game Name">
   ```
3. **Recommended sizes**: 
   - Hero images: 1200x600px
   - Game cards: 400x250px
   - Use GIF format for animated effects
