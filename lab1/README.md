ASSETS FOLDER GUIDE
===================

This folder contains all images for the website.


REQUIRED IMAGE FILES:
---------------------

Homepage images (Hero section):
  - pet-main.jpg    : Main pet image in center (large circle)
  - pet-left.jpg    : Small pet image on bottom left
  - pet-right.jpg   : Small pet image on bottom right

Blog page images:
  - blog-1.jpg      : Senior pet care tips
  - blog-2.jpg      : Grooming at home
  - blog-3.jpg      : Pet nutrition essentials
  - blog-4.jpg      : Exercise and play
  - blog-5.jpg      : Health checkup guide
  - blog-6.jpg      : Building strong bonds with pets


IMAGE REQUIREMENTS:
-------------------

Format:
  - Accepts .jpg or .png files
  - Filename must be lowercase with hyphens

Dimensions:
  - Hero images: 500x500 pixels or larger (square works best)
  - Blog images: 800x450 pixels or larger (landscape orientation)

File size:
  - Keep under 2MB per image for faster loading


IMPORTANT NOTES:
----------------

After adding or replacing images:
  1. Save the file in this folder
  2. Check that the filename matches exactly
  3. Open your browser and press F5 (or Cmd+R on Mac) to refresh
  4. If no image exists, the website will show emoji placeholders

Filename must match exactly. Examples:
  - Correct: pet-main.jpg
  - Wrong: Pet-Main.jpg, petmain.jpg, pet_main.jpg
