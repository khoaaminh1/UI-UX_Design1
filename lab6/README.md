# Lab 6: Responsive Restaurant Landing Page

## Project Overview

This project is a responsive, single page website for a modern restaurant named "Foodie". The goal was to build a functional and visually appealing website based on common design patterns, using Bootstrap and custom JavaScript for interactivity.

## Technologies Used

*   HTML5
*   CSS3
*   Bootstrap 5
*   JavaScript (for interactive elements)
*   Animate.css (for animations)

## Key Features

*   **Fully Responsive Design:** The layout adapts seamlessly to desktop, tablet, and mobile screen sizes using the Bootstrap grid system.
*   **Interactive Menu Reveal:** Users can click a button to reveal featured dishes one by one, creating an engaging discovery experience.
*   **Animation Effects:** The site utilizes Animate.css to provide smooth, attention grabbing animations for menu items.
*   **Booking Modal:** The reservation form is placed inside a Bootstrap modal (pop up window), which can be triggered from multiple locations on the page for user convenience.
*   **Single Page Navigation:** A fixed navigation bar allows users to smoothly scroll to different sections of the page.

## How to View

To view the website, you can open the `index.html` file directly in your web browser. For the best experience, it is recommended to use a live server.
