# Icons for FoodHub

Place your custom icons here (SVG or PNG format):

- `home.svg` or `home.png` - Home icon
- `search.svg` or `search.png` - Search icon
- `orders.svg` or `orders.png` - Orders icon
- `profile.svg` or `profile.png` - Profile icon

Or you can use emoji/Unicode as currently done in the app.

## How to use:
1. Create or download your icons (recommend SVG for scalability)
2. Place them in this folder
3. Update `index.html` to reference them:
   ```html
   <span class="nav-icon"><img src="assets/icons/home.svg" alt="Home"></span>
   ```

Current icons use emoji, so this folder is optional if you prefer emoji.
