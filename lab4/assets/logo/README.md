# FoodHub Logo

Place your FoodHub logo here (SVG or PNG format):

- `foodhub-logo.png` or `foodhub-logo.svg` - Logo for header (recommend 40x40px or higher)

## How to use:
Once you add the logo image, update `index.html` line in the header:

Change this:
```html
<h1>FoodHub</h1>
```

To this:
```html
<img src="assets/logo/foodhub-logo.png" alt="FoodHub" class="logo">
<h1>FoodHub</h1>
```

Or replace the cart icon (🛒) with the logo by updating the header-right section.
